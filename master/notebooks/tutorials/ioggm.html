‚ÄúInstructed OGGM‚Äù: running IGM within the OGGM workflow#

This tutorial gives you the tools to run IGM within OGGM and also compare it with OGGM runs.

This is very much work in progress. You‚Äôll need an IGM installation for this to run. The notebook currently does not run on OGGM Hub, because of the Tensorflow depedency. We are working on it!
